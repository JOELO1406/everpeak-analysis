# everpeak-analysis
### EverPeak Retail Analysis
### Este proyecto realiza un análisis integral de datos (Business Intelligence) sobre el conjunto de datos de operaciones comerciales de EverPeak. El objetivo principal es auditar la calidad de la información, entender el comportamiento del consumidor y evaluar la eficiencia operativa de los puntos de venta para respaldar la toma de decisiones estratégicas.
📋 Descripción del Proyecto
A través de este análisis, se abordaron múltiples dimensiones del negocio utilizando un enfoque guiado por datos:

Calidad y Limpieza de Datos: Se identificaron y controlaron valores nulos o faltantes en variables críticas como métodos de pago y cantidades. Además, se limpiaron y auditaron los datos utilizando métodos estadísticos como el Rango Intercuartílico (IQR) y Z-score para aislar y tratar valores atípicos (outliers).

Eficiencia Comercial por Departamento: Se evaluó la productividad del piso de venta analizando las ventas por metro cuadrado y el porcentaje de participación por departamento. Esto permitió identificar categorías de alto rendimiento (como productos básicos y alimentos frescos) frente a secciones con bajo desempeño operativo (como jardinería y artículos de oficina).

Segmentación de Clientes: Se desarrollaron funciones de segmentación avanzadas para clasificar las transacciones por volumen de compra (alto vs. bajo volumen) en combinación con las preferencias en los métodos de pago, facilitando el diseño de estrategias comerciales dirigidas.

Análisis Geográfico: Se examinó la distribución del rendimiento de ventas a nivel regional, profundizando en el volumen de artículos y transacciones en mercados clave como la región Sur.

🛠️ Tecnologías Utilizadas
